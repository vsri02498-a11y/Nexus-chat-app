# Backend Setup for Nexus Chat App

## Prerequisites
- Node.js (version x.x.x)
- MongoDB (or your choice of database)

## Installation Steps
1. Clone the repository.
   ```
   git clone https://github.com/vsri02498-a11y/Nexus-chat-app.git
   ```
2. Navigate to the project directory.
   ```
   cd Nexus-chat-app
   ```
3. Install dependencies.
   ```
   npm install
   ```

## Running the Application
- Start the server:
   ```
   npm start
   ```
